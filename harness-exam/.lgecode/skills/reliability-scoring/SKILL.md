# 신뢰도 평가 (3축, 총 30점)

## 평가 축

| 축 | 10점 | 7점 | 4점 | 0점 |
|----|------|-----|-----|-----|
| 출처 신뢰도 | 전문매체(McKinsey,Reuters) | 경제지/뉴스 | 블로그/커뮤니티 | 출처불명 |
| B2B 관련성 | 직접 B2B 거래 | 공급망 영향 | 간접(기술/정책) | B2B 무관 |
| 실행 가능성 | 즉시 액션(1~2주) | 1개월 내 반영 | 분기/중기 참고 | 액션 불가 |

## 등급

A(25~30): 우선 포함 / B(18~24): 포함 권장 / C(10~17): 대체재 없으면 포함 / D(0~9): 제외

## 출력 형식

```
항목: {제목}
- 출처:{X}/10, 관련성:{X}/10, 실행:{X}/10 → 총점:{X}/30 (등급:{A~D})
```
