# B2B 리포트 디자인 시스템

## 고정 레이아웃

```
┌─────────────────────────────────┐
│  HEADER: 제목 + 날짜 + 메타     │  배경: 다크(#1E293B)
├─────────────────────────────────┤
│  HERO: 핵심 인사이트 2~3개 카드  │  배경: 그라데이션(블루)
├─────────────────────────────────┤
│  STATS: KPI 수치 3~4개          │  배경: 흰색 카드
├─────────────────────────────────┤
│  ITEMS: 5개 상세 카드            │  1컬럼 스택
├─────────────────────────────────┤
│  FOOTER: 면책 + 생성 메타        │  배경: 밝은 회색
└─────────────────────────────────┘
```

---

## 색상 시스템

```
페이지 배경: #F8FAFC
카드 배경: #FFFFFF
테두리: #E2E8F0
텍스트(제목): #1E293B
텍스트(본문): #475569
텍스트(캡션): #94A3B8

점수 색상:
- 25~30점: #22C55E (초록)
- 18~24점: #F59E0B (주황)
- 0~17점: #EF4444 (빨강)

카테고리 색상:
- 시장 동향: #3B82F6 (Blue)
- OEM 전략: #8B5CF6 (Purple)
- 기술 트렌드: #06B6D4 (Cyan)
- 경쟁/정책: #F59E0B (Amber)
```

---

## 타이포그래피

```
폰트: system-ui, sans-serif
H1: 24px, Bold (제목)
H2: 20px, Semibold (섹션)
H3: 17px, Semibold (카드 제목)
Body: 15px, Regular (본문)
Caption: 13px, Regular (메타)
Badge: 12px, Medium (태그)
Stat: 28px, Bold (수치)
```

---

## 컴포넌트

### Hero Card
- 배경: linear-gradient(135deg, #1E40AF, #3B82F6)
- 텍스트: 흰색
- 패딩: 20px
- 모서리: 12px
- 내용: 핵심 문장 + 점수

### Item Card
- 배경: 흰색
- 테두리: 1px solid #E2E8F0
- 왼쪽 보더: 4px solid [카테고리 색상]
- 패딩: 20px
- 모서리: 8px
- 내용: 순위 + 카테고리태그 + 제목 + 요약 + 시사점 + 출처

### Stats Card
- 배경: 흰색
- 테두리: 1px solid #E2E8F0
- 텍스트 중앙정렬
- 수치(큰 숫자) + 라벨(작은 텍스트)

### 시사점 박스
- 배경: #F8FAFC
- 왼쪽 보더: 3px solid #3B82F6
- 패딩: 12px
- What / So What / Now What(Bold)

---

## 반응형 (2단계)

- **데스크톱** (≥768px): max-width 900px, Hero 카드 가로 배치
- **모바일** (<768px): 1컬럼, 패딩 축소, 폰트 크기 감소
