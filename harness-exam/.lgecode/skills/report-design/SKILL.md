# 리포트 디자인 시스템

## 레이아웃 (고정)

1. HEADER: 제목+날짜 (배경 #1E293B)
2. HERO: 핵심 인사이트 2~3개 카드 (그라데이션 블루)
3. STATS: KPI 수치 3~4개 (흰색 카드)
4. ITEMS: 3개 상세 카드 (1컬럼)
5. FOOTER: 면책+메타 (밝은 회색)

## 색상

| 용도 | 값 |
|------|-----|
| 배경 | #F8FAFC |
| 카드 | #FFFFFF |
| 테두리 | #E2E8F0 |
| 텍스트(제목) | #1E293B |
| 텍스트(본문) | #475569 |
| 텍스트(캡션) | #94A3B8 |

**점수**: 25~30 #22C55E(초록), 18~24 #F59E0B(주황), 0~17 #EF4444(빨강)

**카테고리 왼쪽보더**: 시장동향 #3B82F6, OEM전략 #8B5CF6, 기술트렌드 #06B6D4, 경쟁/정책 #F59E0B

## 타이포

폰트: system-ui, sans-serif. H1 24px Bold, H3 17px Semibold, Body 15px, Caption 13px, Stat 28px Bold, Badge 12px.

## 간격

섹션간 32px, 카드간 16px, 카드패딩 20px, 요소간 12px.

## 제약

- 외부 CDN/이미지/웹폰트 금지, emoji만 사용
- 단일 HTML, 150KB 이하
- 반응형: ≥768px 데스크톱 / <768px 모바일
