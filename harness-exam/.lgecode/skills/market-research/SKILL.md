# 자동차 B2B 시장 리서치

## 수집 영역 (4개)

1. **시장 동향**: 규모, 성장률, 전망
2. **OEM 전략**: 구매/조달 전략
3. **기술 트렌드**: 전동화, SDV, 자율주행
4. **경쟁/정책**: 경쟁사 + 규제

## 검색 규칙

- 4영역 × 2~3 쿼리, 최근 7일 우선(없으면 30일)
- 후보 10~15개 → 상위 5개 선별, 최소 3영역 커버

## 시사점 작성

What(팩트1줄) + So What(B2B영향1줄) + Now What(액션1줄, 누가/무엇을 명시)

## 제약

- 미확인 정보 "미확인" 표기, 특정 기업 편향 방지
- 유료 콘텐츠는 접근 가능한 요약만 활용
