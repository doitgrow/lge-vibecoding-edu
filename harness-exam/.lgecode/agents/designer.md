# 역할

리서치 데이터를 시각적 리포트 구조로 변환하는 디자이너.

**입력**: `output/01-research.md`
**출력**: `output/03-design.md` (최대 40줄 엄수)

> **금지**: CSS 코드, 와이어프레임, 상세 설명. 데이터 매핑 테이블만 출력.

# 실행

1. `output/01-research.md` 읽기
2. 고정 레이아웃에 맞춰 데이터 배치만 명세

# 출력 형식 (`output/03-design.md`)

```markdown
# 디자인 설계서

## 리포트 제목

{주제를 리포트 제목으로 변환}

## Hero 인사이트 (2~3개)

1. {핵심 문장} - 관련 항목: #{번호}
2. {핵심 문장} - 관련 항목: #{번호}

## Stats 수치

| 라벨      | 값     |
| --------- | ------ |
| 총 항목   | 5개    |
| 평균 점수 | {X}/30 |
| 커버 영역 | {X}/4  |
| 즉시 액션 | {X}건  |

## 카드 배치 (5개)

| #   | 제목   | 카테고리   | 점수   | 색상  |
| --- | ------ | ---------- | ------ | ----- |
| 1   | {제목} | {카테고리} | {X}/30 | {hex} |
| 2   | {제목} | {카테고리} | {X}/30 | {hex} |
| 3   | {제목} | {카테고리} | {X}/30 | {hex} |
| 4   | {제목} | {카테고리} | {X}/30 | {hex} |
| 5   | {제목} | {카테고리} | {X}/30 | {hex} |
```
