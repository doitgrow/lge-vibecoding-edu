## 워크플로우

1. 파일을 Read 도구로 읽는다.
2. 텍스트에서 다음 정보를 파싱한다:
   - 이름 (name)
   - 나이 (age) — 숫자형으로 변환
   - 직업 (job)
   - 취미 (hobbies) — 배열로 변환
3. 파싱한 정보를 아래 JSON 스키마에 맞게 변환한다.
4. 결과 JSON을 사용자에게 출력한다.

## 출력 JSON 스키마

```json
{
  "name": "string",
  "age": number,
  "job": "string",
  "hobbies": ["string", "..."]
}
```

## 예시

### 입력 (sample.txt)
```
안녕하세요. 제 이름은 김영희이구요, 나이는 22세입니다.
현재 하고 있는 일은 디자이너이며, 취미는 독서와 낚시입니다.
```

### 출력
```json
{
  "name": "김영희",
  "age": 22,
  "job": "디자이너",
  "hobbies": ["독서", "낚시"]
}
```

## 파싱 규칙

- **이름**: `이름은 {이름}이구요` 또는 `이름은 {이름}입니다` 패턴에서 추출
- **나이**: `나이는 {숫자}세` 패턴에서 숫자만 추출하여 정수로 변환
- **직업**: `하고 있는 일은 {직업}이며` 또는 `직업은 {직업}` 패턴에서 추출
- **취미**: `취미는 {취미1}와 {취미2}` 패턴에서 `와/과/,` 기준으로 분리하여 배열로 변환
